Monkey Run is a game created by the GameMaker software.
This game is available on Windows and IOS.
The character in this game is a monkey, with the goal
of collecting coins while dodging obstacles.
The instructions of the game are pretty simple. To move forward,
hold the right arrow. -> To jump, click the space bar. __ The goal is to
collect as many coins as possible. The best way to dodge obstacles
is to jump over them!
